AppSales Import / Export

You can import report files that you downloaded manually from iTunes Connect or with a different application to AppSales.

Report files must be stored in a zip archive without subdirectories and have a txt extension. Only daily and weekly reports are supported.

These are the original report files that you downloaded with AppSales. Please note that older versions of AppSales did not store the original report files, so that this list may not be complete.
